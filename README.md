Subspace Relay uses Windows’ built‑in Bluetooth COM ports. To configure it:

A. Open Windows Bluetooth COM Port Settings
Go to Settings → Bluetooth & devices

  1. Scroll down and click More Bluetooth settings

  2. In the popup window, open the COM Ports tab

  3. Look for COM ports labeled Outgoing

  4. Insert and remove the brackets where the script says "insert outgoing "
